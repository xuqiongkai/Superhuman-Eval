# Superhuman-Eval

The project for ***Humanly Certifying Superhuman Classifiers*** (ICLR 2023, spotlight).

### Installation
**Requirement**: PyTorch, Numpy, etc.
**Download**:
```
git clone https://github.com/xuqiongkai/Superhuman-Eval.git
```

### Confidence Score
```
python fsa_example.py
```
